TOC Curve Generator

Zhen Liu
Clark Univeristy
zhenliu3@clarku.edu

TOC Curve Generator is a computer program to make TOC curves. Users can generate TOC curves from various file formats and customize TOC curves for research.

How to Download and Run TOC Curve Generator

Click the right top button "Download" to download the program. Extact files to the local computer. Click "TOC_curve_generator.exe" to start the progam. The instruction slide is in the folder "instruction".
